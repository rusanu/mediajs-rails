media-rails
===========

Rails asset gem for JQuery Media plugin


INSTALL
=======

    gem 'mediajs-rails',  :git => 'git://github.com/rusanu/mediajs-rails.git'

USE
===

Add to application.js:

    //= require jquery.media

Lear More
=========

Visit [JQuery.Media](http://jquery.malsup.com/media/).
